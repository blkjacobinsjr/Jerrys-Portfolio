In the new AI driven age- if machines can talk why cant our stories? So myself and a good friend Adrian Padurean developed this Chrome Extension MVP detects the show you're watching and provides access to via a chat ui that brings all the characters alive. Effectively making it possible to talk to 1m+ AI characters.
